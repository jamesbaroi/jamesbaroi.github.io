Cellular Automaton in JavaScript

March 16, 2023

A tutorial on how to create Conway's game of life.

Overview

Build John H. Conway's game of life in JavaScript using the HTML canvas element to render graphics on a 200 x 200 cell grid allowing for controls to set the initial seed probability, configure game rules, reduce grid size, and change cell and grid colors. Implement buttons to pause, resume, and restart the game, and output metrics including population count, frame rate, and latency. Click here to preview the project. Scroll down to the end for the complete code.

Prerequisites

This tutorial requires minimal knowledge of HTML, CSS, and JavaScript (JS), the ability to serve these files, and the use of a code editor.

Plan

First create a game area in JS and output it to HTML. Then construct cells to place into the grid, configure the game rules, define the game initialization and loop, and calculate and output metrics. Finally, create configurable settings and button controls.

Area

Create a grid container in html to output the game area.


<!--index.html-->

<body>

  <div id="grid-container"></div>

</body>

Create the game area object in JS, assign to it a grid property that takes in a created canvas element, and setup a starting method. 


/**script.js */

const area = {

  grid: document.createElement('canvas'),

  start() { }
}

Get the grid-container and pass the canvas grid in the start method. Do a quick test by running the method. Open dev tools to check the canvas element that should appear in the grid container. 


const contGrid = document.getElementById('grid-container')

const area = {

  grid: document.createElement('canvas'),

  start() {

    contGrid.insertBefore(this.grid, contGrid.childNodes[0])
  }
}

area.start()

Get the canvas 2d context and assign it to the start method. Create grid dimension properties. 


start() {

  contGrid.insertBefore(this.grid, contGrid.childNodes[0])

  this.context = this.grid.getContext('2d')

  this.grid.width = 2000
  this.grid.height = 2000

  this.cols = 200
  this.rows = 200
}

Add css to visualize the grid perfectly centered on any screen.


body {
  background-color: rgb(32, 33, 36);
  display: grid;
  justify-content: center;
  margin: 0;
  padding: 0;
}

#grid-container {
  background-color: rgb(95, 99, 104);
  box-sizing: border-box;
  margin: 8px;
}

#grid-container > canvas {
  max-height: calc(100vh - 20px);
  max-width: 100%;
