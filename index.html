About
I am a full stack, javascript web developer. Read on to learn about my origin, expertise, and mission.
Origin
I began my journey to become a self taught web developer a few years ago after having completed formal education in the natural sciences and gained years of experience teaching undergradutes in the classroom.
My initial desire to supplement my career as an aspiring scientist and educator, by learning computer programming, ignited a passion for web development that has materialized into this very web page and that continues to this day.
Expertise
I use javascript unavoidably on the front-end and preferably on the back-end with Node.js as the runtime environment, often with the Express.js framework and MySQL or MongoDB as the database.
I have launched applications on Amazon Web Services (AWS) using EC2 instances.
I have sampled front-end frameworks, namely Angular, React, and Bootstrap but prefer to do without them.
I have tinkered with python and php, C++ and C#.
Mission
My mission is to build web applications.
