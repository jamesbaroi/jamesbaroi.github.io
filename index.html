About
I am a full stack, JavaScript web developer. Read on to learn about my origin, expertise, and mission.
Origin
I began my journey to become a web developer a few years ago after having completed formal education in the natural sciences (MS Chemistry - CSU Frenso, BS Environmental Science - UC Riverside).
My initial desire to supplement my career by learning computer programming, ignited a passion for web development that has materialized into this very web page and that continues to this day.
Expertise
I use JavaScript unavoidably on the front-end and preferably on the back-end with Node.js as the runtime environment, often with the Express.js framework and MySQL or MongoDB as the database.
I have launched applications on Amazon Web Services (AWS) using EC2 instances.
